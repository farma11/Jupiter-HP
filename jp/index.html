Jupiter

自動交渉のための統合プラットフォーム

Github ページへ

Jupiterは，自動交渉のためのオープンアーキテクチャとして実装された自動交渉環境です． Jupiterでは，自動交渉エージェントを開発する機能や自動交渉エージェントのテストベッドを提供します． 特に，Python言語を用いて自動交渉エージェントを実装することができるため， scikit-learn，Chainer，及びTensorflowといった機械学習ライブラリを用いることが容易となっています．

Jupiterは現在開発段階です．今後の展望にご期待ください．

Python3

Python 3を用いて自動交渉エージェントが開発できます．

機械学習

自動交渉エージェントの開発に、機械学習ライブラリを容易に用いることができます．

互換性

GENIUS上でJavaを用いて実装された自動交渉エージェントを使うことができます．

インストール方法

JupiterはPyPIにパッケージ登録されているため，pipを用いてインストールをすることができます． 


    # bash
    $ pip install jupiter-negotiation

以下のコマンドを実行することで動作確認ができます． 


    # bash
    $ jupiter --test
